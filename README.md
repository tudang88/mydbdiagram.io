# MyDBDiagram.io

Personal tool similar to dbdiagram.io for database diagramming.

## Project Structure

```
mydbdiagramio/
├── design/          # Design files
├── output/          # Exported files from the tool
└── src/             # Source code
```

## Getting Started

### Prerequisites

- Node.js 18+ (use `.nvmrc` for version management)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Run Production

```bash
npm start
```

## Code Quality

### Linting

```bash
npm run lint
npm run lint:fix
```

### Formatting

```bash
npm run format
npm run format:check
```

### Type Checking

```bash
npm run type-check
```

## Rules & Standards

- **Language**: TypeScript
- **Indentation**: 2 spaces
- **Quotes**: Single quotes
- **Line Endings**: LF (Unix)
- **Code Style**: ESLint + Prettier
- **Strict Mode**: Enabled
